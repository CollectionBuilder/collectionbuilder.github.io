GH Walkthrough

This walkthrough provides steps for creating an example digital collection using preexisting demo metadata, the CollectionBuilder-GH Template, and GitHub Pages.

You will need to create a free GitHub account for this walkthrough, but no software installation is necessary.

SHEETS Walkthrough

This two-part walkthrough provides steps for using SHEETS' built-in development mode to prototype, test, and share a draft of your digital collection site. You also will learn how to create an example project using preexisting demo metadata, the CollectionBuilder-SHEETS Template, and GitHub Pages.

You will need to create a free GitHub account for this walkthrough, but no software installation is necessary.

Go to Walkthrough

CSV Walkthrough

This walkthrough provides steps for creating an example digital collection using preexisting demo metadata, the CollectionBuilder-CSV Template, and the GitHub Actions feature.

You will need to install free, open-source software on your computer and create a free GitHub account for this walkthrough.
